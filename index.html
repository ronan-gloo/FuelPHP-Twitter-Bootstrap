Speed up template coding

This package tries to be simple and intuitive as possible, to save you to write endless html code...

Form::group_open('info')
	Form::label("Username")
	Form::input('username')->prepend('icon-user')
Form::group_close()

<div class="control-group info">
	<label class="control-label" for="form_">Username</label>
	<div class="controls">
		<div class="input-prepend">
			<span class="add-on">
				<i class="icon icon-user"></i>
			</span>
			<input name="username" value="" type="text" id="form_username">
		</div>
	</div>
</div>

Global configuration in config/bootstrap/bootstrap.php

  • utilities: values for specific bootstrap css classes.
  • strict: checks for atttribute validity, and throw an exception if not.
  • prettyprint: prettyprint the output (will cause huge performance hints)
